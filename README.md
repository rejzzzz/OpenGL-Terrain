#### steps to run the code in wsl:
- pre-requisites: gcc compiler for cpp & ```sudo apt install libglfw3-dev libglew-dev libglm-dev```
1. build the code in wsl
```
g++ <file_name.cpp> -o <build_name> -lglfw -lGLEW -lGL
```
2. run the build
```
./<build_name>
```
